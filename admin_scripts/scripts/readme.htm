SCRIPTS DESCRIPTION

STORAGE:

    TABLESPACES:
        boundobj - Report on objects with extents bounded by freespace
        coalesce - Coalesces all tablespaces in a database (7.3)
        free_ext - Free space map for tablespaces in a database
        free_sp  - Provide information on free space in a database
        ts_free  - Free space in tablespaces (summary)
        ts_ext   - Phisycal placement of segment extents in a tablespace
        ts_seg   - List of users and their segments in a tablespace
        ts_usr   - Owner-to-tablespace location map
        usr_ts   - Tablespace-to-owner location map
        coalescd - Coalesced extents per tablespace
        ts_used  - Tablespace storage information
        ts_defst - Default storage for each of the tablespaces
        quotas   - Database used space report
        systemts - Who sits on the tablespace SYSTEM?
        ts_size  - Tablespaces sizing and using
        ts_rct   - Script for (re)creating tablespaces
        ts_frag  - Tablespaces fragmentation information
        usr_used - Database usage by user and tablespace
        ts_io    - Displays the datafile I/O by tablespace
        frag     - Displays database fragmentation chart

    DATA FILES:
        file_io  - Input/Output statistics by datafile
        io       - I/O distribution by file, tablespace, or device
        phys_io  - Information on physical datafile reads and writes
        ts_io    - Displays the datafile I/O by tablespace
        files    - All database files (control, redo, data etc.)
        datafile - Document datafile sizes and locations
        ts_file  - Datafiles for tablespaces (sizing info)
        filefrag - Free space fragmentation in datafiles
        file_ext - Show datafile AUTOEXTEND characteristics
        filerows - Table's rows per datafile (striping effectiveness)

    SEGMENTS:
        unused   - Determine space allocation within a segment (7.3)
        seg_ext  - Segment extents (an interactive script)
        seg_type - List segments grouped by segment type
        ts_ext   - Phisycal placement of segment extents in a tablespace
        ts_seg   - List of users and their segments in a tablespace
        ts_usr   - Owner-to-tablespace location map
        usr_ts   - Tablespace-to-owner location map
        boundobj - Report on objects with extents bounded by freespace
        cant_ext - Unextendable database segments
        not_extd - Database objects that are not be able to extent
        max_ext  - Objects with extents allocated close to Max_Extents
        many_ext - Segments with more then specified number of extents
        my_space - Find space used for one's own segments
        blokrows - Number of rows per table block
        tab_size - Calculates size of a table
        systemts - Who sits on the tablespace SYSTEM?
        seg_cols - Physical order of columns in a table
        corrupt  - Find the segment having the file and block numbers
        dealloc  - Deallocate unused space from (table) segments
        t&i_size - Table and their index segment sizes

    ROLLBACKS:
        rbs      - Rollback segments general information
        rbs_usr  - Rollback segments, active users and SQL statements
        rbs_cont - Rollback segments contention
        rbs_usag - Rollback segments - Transactions - Statistics
        rbs_wait - Rollback segment wait statistics
        tran_tbl - Number of rollbacks performed on the transaction tables
        rbs_chk  - Checks how effectively the RBS have been set up
        rbs_def  - Rollback default storage parameters
        rbs_rct  - Script for (re)creating rollback segments
        rbs_tx   - Rollback segments in use
        rbs_off  - Set available rollback segments offline
        rbs_on   - Set available rollback segments online
        rbs_stat - Rollback segments statistics
        rbs_life - Rollback segments life's watcher (a demo script)
 

SECURITY:

    USERS:
        usr_rct  - Create a script to recreate users
        users    - Oracle users report (summary)
        id       - Display current username and database
        set_ts   - Set user tablespace defaults (by user name or role)
        rm_usr   - Drop users (by name or role)
        like_usr - Create an ORACLE user account like another account
        passwd   - Change current user's password
        set_pwd  - Set the "standard" user password (by user name or role)
        protect  - Mass change password for group of users

    PROFILES:
        used_sp  - Database used space report (to check quotas)
        quota    - Report how much space each user has (quota usage)
        profiles - Profiles report
        quotas   - Users space quotas by users
        usr_used - Database usage by user and tablespace (to check quotas)

    ROLES AND PRIVILEGES:
        privs    - Privileges granted to a user or role (all-purpose)
        tab_grnt - Create output that can recreate all table(s) grants
        sec_obj  - Direct object grants to users and roles
        sec_priv - Direct system privilege grants to users and roles
        sec_role - Direct role grants to users and roles
        monitor  - Grants to user who will be monitoring
        grnt_sys - Grants select on SYS tables to users
        rvk_sys  - Revokes select on SYS tables from a user
        uprivcrt - The script to create the DBA_USER_PRIVS view
        usr_priv - Which privileges are available to a given user
        crud     - Access privileges to objects in CRUD chart by role
        grnt_obj - Grant privileges on objects
        noaccess - Objects which have no grants
        roles    - List roles

INSTANCE:

    SGA:

        BUFFER CACHE:
            hit_rat1 - Cumulative hit ratio
            hit_rat2 - Cumulative hit ratio
            usr_hit1 - Hit ratio by users
            usr_hit2 - Hit ratio by users (Mark Gurry)
            bufcache - Buffer cache usage (general and detailed reports)
            buf_gets - All statements that are taking longer than
                       2 seconds to scan through the buffer cache
            buf_wait - Number of waits for free buffer
            buf_free - Database buffer cache free and used blocks
            buf_chk  - Database block buffer cache summary
            sga_dec  - Examine statistics in the X$KCBCBH table with
                       intent to shrink the SGA
            sga_inc  - Examine statistics in the X$KCBRBH table with
                       intent to expand the SGA
            lru_stat - Buffer cache statistics in X$KCBRBH and X$KCBCBH
            dirty    - Information about dirty buffers in the buffer cache

        SHARED POOL:
            sh_pool  - Shared pool summary
            bad_obj  - Report on "bad" objects in the shared pool
            pinned   - Query the SGA to determine pinned PL/SQL objects
            pin      - Pin packages in SGA (an interactive script)
            unpin    - Unpin packages in SGA (an interactive script)
            pin_sys  - Pin some SYS packages in SGA (a batch script)
            cursors  - Display open cursors (summary by user)
            poor_sql - "Poorly performing" SQL statements
            sql_read - Statements in shared pool with the most disk reads
            most     - Most consumed program objects in the shared pool
            poolfail - Shared pool reserved size recommendation
            poolsize - Recommended shared pool size
            pool_adv - Calculates size of shared pool based on # of users
            res_pool - Reserved pool settings for the shared pool area
            usr_curs - User session cursor usage
            free_mem - Shared pool free memory
            poolload - Shared pool load level
            pool_chk - Integrated information about the shared pool
            libcache - Library cache report
            poolstat - Shared pool stored object statistics
            poolutil - Estimates shared pool utilization
            sql_mem  - SQL area memory usage showing SQL text and memory
                       categories
            last_sql - Last executed SQL statement by session with stats

        OTHER:
            up_time  - The time when an instance was started
            dirt_que - The average length of the write request queue
            sga_stat - Key SGA memory usage
            recursiv - Number of recursive calls that the instance has
                       performed
            dc       - Dictionary cache statistics
            dc_eff   - Dictionary cache effectiveness
            ctl_file - Document control file location and status
            eventset - Reports all events set in the current session and
                       their level
            hits     - Primary ORACLE internal hit ratios and memory
            sga_hits - SGA cache hit ratios
            pga      - Display user PGA memory usage
            uga      - Display user UGA memory usage
            pq       - Show Parallel Query vital stats

    INITIALIZATION:
        pars     - INIT.ORA parameters
        undocini - Undocumented INIT.ORA parameters (7.3)
        init_chk - INIT.ORA parameters settings adviser
        init_ora - Recreate the instance INIT.ORA file
        ver      - Show product version information and installed options

    TRANSACTIONS:
        rbs_usr  - Rollback segments, active users and SQL statements
        rbs_act  - Rollback segment usage by active transactions
        transact - Active transactions

    SESSIONS AND PROCESSES:
        ses_mem  - Session memory usage
        my_sess  - Identification of the current session
        my_waits - Current session wait events
        access   - Database objects accessed by users
        waiters1 - Waiting and holding sessions
        waiters2 - Report on sessions waiting for locks
        ses_cpu  - CPU used per session
        tot_wait - Session waits (total)
        ses_io   - Display IO and hit ratio by session
        usr_ses  - Show users session information
        usr_curs - User session cursor usage
        ses_wait - Sessions (users) waits
        lockers  - Sessions blocking other sessions
        ses_evnt - Generate a report on session events by user
        login_tm - User login times
        os_usr   - Current oracle system users
        process  - Current oracle processes
        jobs     - Lists all jobs in the database
        idle_ses - Finding out the time for which sessions has been idle
        eventset - Reports all events set in the current session
        bg       - Activated backgroung processes
        app_info - Application information for sessions

        trace    - Set SQL_TRACE on/off in selected user sessions
        uga      - Display user UGA memory usage
        userenv  - Show values of all USERENV variables
        who      - Who are working?

NOTES

    1. If you use UNIX to run this scripts, you must rename service
    scripts, beginning with underscore "_", to upper case (for example,
    _begin.sql to _BEGIN.sql). File name extention (.sql) must be in lower
    case.
